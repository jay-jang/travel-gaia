> An umbrella term for the various short codes assigned and administered by the International Air Transport Association (IATA), most commonly the 3-letter airport/city location code and the 2-character airline designator. The exact meaning depends on context (airport, airline, or other IATA-managed code).

Because IATA maintains several distinct coding schemes, 'IATA Code' is context-dependent: a 3-letter IATA location code identifies an airport or city served by air transport, while a 2-character IATA airline designator identifies an air carrier. IATA codes are optimized for commercial and consumer-facing use (tickets, schedules, displays) and are not globally unique in the way ICAO codes are. The term is widely used loosely, so precise documentation should specify which IATA code is meant.

**한국어 / Korean** — **IATA 코드** — 국제항공운송협회(IATA)가 부여하고 관리하는 다양한 단축 코드를 포괄적으로 일컫는 용어로, 가장 흔하게는 3자리 공항/도시 위치 코드와 2자리 항공사 지정 코드를 가리킨다. 정확한 의미는 문맥(공항, 항공사, 또는 IATA가 관리하는 기타 코드)에 따라 달라진다.

IATA는 여러 개의 별개 코드 체계를 운영하기 때문에 'IATA 코드'라는 표현은 문맥에 따라 달라진다. 3자리 IATA 위치 코드는 항공 운송이 이루어지는 공항이나 도시를 식별하고, 2자리 IATA 항공사 지정 코드는 항공 운송 사업자를 식별한다. IATA 코드는 상업 및 소비자 대상 용도(항공권, 스케줄, 표시 화면)에 최적화되어 있으며, ICAO 코드처럼 전 세계적으로 고유하지는 않다. 이 용어는 폭넓고 느슨하게 사용되므로, 정확성을 요하는 문서에서는 어떤 IATA 코드를 의미하는지 명시해야 한다.

**Aliases:** `IATA Two-Letter Code`, `IATA Designator`, `IATA Location Identifier`

# Distinctions
- **IATA Code** vs [ICAO Code](/codes/icao-code.md) — IATA codes are short commercial codes (3-letter airport, 2-char airline); ICAO codes are longer (4-letter airport, 3-letter airline) and globally unique for operational/ATC use.
