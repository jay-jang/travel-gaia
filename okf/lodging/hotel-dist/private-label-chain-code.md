> A private label chain code is a dedicated 2-character GDS chain code that an independent hotel or group obtains — typically via a technology partner such as Sabre or a representation company like HotelREZ — so its properties appear in the GDS under their own brand identity rather than buried inside a representation company's shared chain code. It is a specific GDS-identity mechanism, distinct from the generic chain code concept.

Independent hotels usually reach the GDS through a hotel representation company, sitting under that rep's shared 2-character chain code (e.g. HotelREZ's 'HO'). A private label chain code instead gives the hotel or group its own dedicated chain code (e.g. Dorint on 'DO' alongside HotelREZ's 'HO'), powered by the rep's connectivity/CRS — often on Sabre Hospitality technology. The benefit is brand recognition and easier discovery among the travel-agency community, while the group still leverages the rep's GDS connectivity, consortia, and corporate-RFP reach. It is distinct from the generic Chain Code (the 2-character GDS code that groups properties under a brand) in that it specifically denotes an independent obtaining its OWN code rather than using a shared/representation code, and it overlaps operationally with the hotel representation company model.

**한국어 / Korean** — **프라이빗 라벨 체인 코드 (Private Label Chain Code)** — 프라이빗 라벨 체인 코드는 독립 호텔이나 그룹이 (보통 Sabre 같은 기술 파트너나 HotelREZ 같은 대표 회사를 통해) 확보하는 전용 2자리 GDS 체인 코드로, 자사 프로퍼티가 대표 회사의 공유 체인 코드 안에 묻히지 않고 자체 브랜드 정체성으로 GDS에 표시되게 한다. 이는 일반적 chain code 개념과 구별되는 구체적인 GDS 정체성 메커니즘이다.

독립 호텔은 보통 호텔 대표 회사를 통해 GDS에 접근하며, 그 대표사의 공유 2자리 체인 코드(예: HotelREZ의 'HO') 아래에 위치한다. 프라이빗 라벨 체인 코드는 대신 호텔이나 그룹에 자체 전용 체인 코드(예: HotelREZ의 'HO'와 나란히 'DO'를 쓰는 Dorint)를 부여하며, 흔히 Sabre Hospitality 기술 기반으로 대표사의 커넥티비티/CRS가 이를 구동한다. 이점은 브랜드 인지도와 여행사 커뮤니티에서의 검색 편의이며, 그룹은 여전히 대표사의 GDS 연결성·컨소시아·기업 RFP 도달을 활용한다. 이는 일반적 Chain Code(프로퍼티를 한 브랜드로 묶는 2자리 GDS 코드)와 구별되는데, 공유/대표 코드를 쓰는 대신 독립 주체가 자체 코드를 확보한다는 점을 특정하며, 운영상 호텔 대표 회사 모델과 겹친다.

**Aliases:** `Private Label GDS Chain Code`, `Own Chain Code`, `Dedicated Chain Code`

# Distinctions
- **Private Label Chain Code** vs [Chain Code](/common/codes/chain-code.md) — A chain code is the general 2-character GDS code that groups properties under a brand or representation entity; a private label chain code specifically denotes an independent hotel/group obtaining its OWN dedicated code rather than sitting under a representation company's shared code.
- **Private Label Chain Code** vs [Hotel Representation Company](/lodging/hotel-dist/hotel-representation-company.md) — A private-label chain code is the GDS identifier a property is sold under; a representation company is the firm that provides and operates that code (and broader services) for the property.
