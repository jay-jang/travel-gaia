> A digital ticket or voucher delivered to a smartphone (QR/barcode/wallet pass) and redeemed by scanning at the point of service.

Mobile tickets remove the print/exchange step: the reseller issues a scannable code (often a wallet pass) the moment a booking is confirmed, and the operator validates it at a turnstile or with a handheld scanner. This requires the operator's redemption system and the reseller to share a code namespace, which is why mobile ticketing is tightly coupled to API connectivity and instant confirmation.

**한국어 / Korean** — **모바일 티켓(Mobile Ticket)** — 스마트폰으로 전달되어(QR/바코드/월렛 패스) 서비스 현장에서 스캔으로 사용 처리되는 디지털 티켓·바우처다.

모바일 티켓은 인쇄·교환 단계를 없앤다. 예약이 확정되는 즉시 리셀러가 스캔 가능한 코드(흔히 월렛 패스)를 발급하고, 운영사는 개찰구나 휴대 스캐너로 검증한다. 이를 위해 운영사 사용처리 시스템과 리셀러가 코드 네임스페이스를 공유해야 하므로, 모바일 티켓팅은 API 연동·즉시확정과 긴밀히 결합된다.
