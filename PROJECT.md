# Project: Travel Gaia Knowledge Graph & Mindmap Layout Redesign

## Architecture
Travel Gaia is a single-file, offline-compatible glossary and graph visualization application.
- **Source template**: `build/template.html` (contains index HTML and scripts)
- **Output target**: `index.html` (generated by running `node build/build.mjs`)
- **Interactive UI**: D3.js based Knowledge Graph and Mindmap UIs.
- **Bilingual Engine**: Integrates with global `LANG` variable and `t()` translation function.
- **UI Test Suite**: `build/verify-ui.mjs` (validates UI functionality using JSDOM).

## Interface Contracts
- **Routing Endpoints**:
  - `#/graph` (or Knowledge Graph view)
  - `#/mindmap` (or Mindmap view)
- **Translation System**:
  - `LANG` (global): String 'ko' or 'en'.
  - `t(key)`: Returns translated string for the given key.
- **D3 Selection & Graph State**:
  - Nodes: Styled capsule cards with gradients/borders.
  - Links: Solid color parent-child vs thin dashed gray cross-category links.
  - Accordion expansion/collapse behavior on Category click.
  - Orthogonal link routing in horizontal Mindmap.
  - Path glow highlighting from root to selected node.
  - Depth controls ("Collapse All", "Show Categories", "Expand All").
