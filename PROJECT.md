# Project: Travel Gaia Classroom Page Redesign

## Architecture
Travel Gaia is a single-file, offline-compatible glossary and graph visualization application.
- **Source template**: `build/template.html` (contains index HTML and scripts)
- **Output target**: `index.html` (generated by running `node build/build.mjs`)
- **Interactive UI**: SVG-based classroom learning system replacing the legacy Canvas 2D container.
- **Bilingual Engine**: Integrates with global `LANG` variable and `t()` translation function.
- **UI Test Suite**: `build/verify-ui.mjs` (validates UI functionality using JSDOM).

## Interface Contracts
- **Routing Endpoints**:
  - `#/classroom`: Classroom page showing the interactive learning platform.
- **Translation System**:
  - `LANG` (global): String 'ko' or 'en'.
  - `t(key)`: Returns translated string for the given key.
- **DOM Elements**:
  - `.classroom-wrap`: Container wrapper for classroom UI.
  - `#classTitle`: Classroom title.
  - `#classStepCount`: Classroom step count text.
  - `#classNarration`: Classroom narrative text.
  - `#classBtnBack`: Back button.
  - `#classBtnNext`: Next/restart/finish button.
  - `.classroom-tab`: Tab buttons for selection.
